# rn-turkey-city-list

List of cities in Turkey.

## Installation

```sh
npm install rn-turkey-city-list
```

## Usage

```js
import RnTurkeyCityList from "rn-turkey-city-list";

// ...

const result = await RnTurkeyCityList.getCities();
```
